pub const REPEAT_ABOVE: Self

This is an end for the LOOP command. In contrast with the other block ends, most of logic is here. All this does is reset the command index to the top LOOP marker command.

This is likely related to move routes somehow, like how the TEXT_DATA command extends the SHOW_TEXT command. However, I can’t find the implementation of this instruction. Furthermore, I don’t know why it’s even included; Its data always duplicates the data of the SET_MOVEMENT_ROUTE command.
